petulant-spice
==============

A Single Page Todo App with Node and Angular

To install:  

    git clone https://github.com/nicholaschris/petulant-spice.git
    cd petulant-spice
    npm install

To run:  

    node server.js

then navigate to localhost:8080.

Requirements:  
* node and npm installed  
* a mongodb from modulus
